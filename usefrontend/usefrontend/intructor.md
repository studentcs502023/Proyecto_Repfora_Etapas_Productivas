


 1. Búsqueda del aprendiz  Permitir buscar por primer nombre y primer apellido, por documento de identidad o ficha. Dependiendo del aprendiz que sea seleccionado, se mostrará la cantidad de seguimientos:   TRES PARA NIVEL TECNICO Y TECNOLOGO.  DOS PARA EL NIVEL OPERARIO.  2. Selección del número de seguimiento  Indicará el número de seguimiento que se desea subir, además de la opción de ""Seguimiento extraordinario"" pero para este se requiere de un permiso del administrador. (Explicado en la HU-9)  Si ese seguimiento ya fue cargado anteriormente, el sistema mostrará una advertencia indicando que ya existe, bloqueará los campos siguientes y no permitirá continuar.  Si no existe un archivo cargado para ese seguimiento, se habilitará el siguiente paso.  3. Subida del archivo  Permitir cargar un archivo en formato PDF con un peso máximo de 10 MB.  4. Validación con IA  Debajo del área de carga del PDF, se mostrará un botón ""Verificar con IA"". Si no se presiona este botón, no será posible continuar y si la validación indica que el archivo es incorrecto, tendrá que volver a intentar.  Si la validación es correcta, se habilitará el botón ""Confirmar"".
